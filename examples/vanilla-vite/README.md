# Listedb SyncManager Vanilla Vite Example

This example demonstrates a simple collaborative text editor using the Listedb SyncManager with a vanilla TypeScript and Vite setup.

## Setup

1.  **Configure Firebase:**
    *   Open `src/firebase-config.ts`.
    *   Replace the placeholder values with your own Firebase project's configuration. You can find this in your Firebase project's settings.

2.  **Install Dependencies:**
    ```bash
    npm install
    ```

## Running the Example

1.  **Start the Vite development server:**
    ```bash
    npm run dev
    ```

2.  Open your browser to the URL provided by Vite (usually `http://localhost:5173`).

3.  Open the same URL in another browser tab or on another computer to see the real-time collaboration in action.
